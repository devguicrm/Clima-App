# ClimaNow — Weather App

Aplicação web responsiva para consulta de clima atual e previsão dos próximos dias.

## 🔗 Acesse o projeto

[Ver site publicado](https://devguicrm.github.io/Clima-App/)

## 📌 Sobre o projeto

O **ClimaNow** é um projeto front-end desenvolvido para consultar o clima de cidades em tempo real. A aplicação permite pesquisar uma cidade, usar a localização atual, visualizar temperatura, sensação térmica, umidade, vento, precipitação, índice UV, previsão semanal e temperatura por horário.

## 🚀 Funcionalidades

- Busca de clima por cidade;
- Uso da localização atual do usuário;
- Cidades rápidas para consulta;
- Clima atual com ícone e descrição;
- Sensação térmica, umidade, vento, chuva e UV;
- Previsão dos próximos dias;
- Temperatura por horário;
- Tema claro e escuro;
- Salvamento do tema no LocalStorage;
- Interface responsiva.

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- LocalStorage
- API Open-Meteo
- Font Awesome
- Google Fonts

## 🌦️ API utilizada

O projeto usa a API pública da Open-Meteo para previsão do tempo e geocodificação de cidades. A API não exige chave de autenticação.
